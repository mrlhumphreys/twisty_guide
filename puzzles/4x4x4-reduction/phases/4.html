Parity issues

Sometimes some odd cases can appear. Edges might need twisted around.

Case 1

Only one edge not oriented correctly. Break up and pair up edge.

(r2 B2 U2 l) (U2 r' U2 r) (U2 F2 r F2) (l' B2 r2)

Case 2

Only two corners not oriented correctly. Break up and pair up edge. Then solve again as a 3x3.

r2 U2 r2 U2 u2 r2 u2
